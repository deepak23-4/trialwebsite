Attribute selectors




Attribute selector is used to select HTML element based on their attribute name and attribute value
In CSS, you can use attribute selector to target HTML elements with certain attributes
Attribute selectors are case sensitive by default and are written inside brackets [    ]
They all start with the attribute name and end with an equal sign followed by the attribute value usually in quote. What goes in between the attribute name and equal sign is what makes the difference among the selectors

Syntax: 
      selector[attribute-expression]{
        CSS declaration list;
      }
where, selector is optional, it can be a tag, class, id selector etc,
attribute expression can be a valid combination of attribute name, value and associated operators.
There are seven different types of selectors we can find and each one with a different syntax and they are :
  1. [attribute-name] selector
  2. [attribute-name="attribute-value"] selector
  3. [attribute-name^="attribute-value"] selector
  4. [attribute-name~="attribute-value"] selector
  5. [attribute-name*="attribute-value"] selector
  6. [attribute-name|="attribute-value"] selector
  7. [attribute-name$="attribute-value"] selector


[attribute-name]

The [attribute-name] selector is used to select HTML element with a specified attribute or in other words it selects any HTML element which has an attribute name in it.
Example: 
      [align]{
        border: 2px solid red;
      }
It selects any HTML element targeted by the attribute name "align" and apply 2px solid red color border.


[attribute-name="attribute-value"]

The [attribute-name="attribute-value"] selector is used when we need to specify a particular HTML element or we can say when we don't want to target all the html element with the same attribute value. In order to target a particular HTML element we can use a specific selector like tag selector, id selector, class selector etc 
      p[align="right"]{
        background-color:blue;
      }
It selects a <p> tag and targets an attribute "align" with a value right and style it with a background color blue.


[attribute-name^="attribute-value"] : Caret sign indicates "begin with"

The [attribute-name^=attribute-value} selector is used to select HTML elements whose attribute value begins with a specified value.
In other words, it selects any HTML elements targeted by the selector which contains an attribute name and whose value begins with the value specified in attribute expression

Example 1: 
      a[href^="https://"]{
        border= 2px solid blue;
      }
In this example, it selects any anchor tag which has a hyperlink "href" and whose value "begins with" https:// Example 2: 
      img[alt^="art"]{
        border- 2px solid green
      }
here in example 2 it will select images with alt text "art show" or "artistic pattern" but not an image with the alt text "Arthur" or "Art" because "A" begins with a capital letter and Attribute selector is case sensitive.

[attribute-name$="attribute-value"] : Dollar sign indicates "ends with"


The [attribute-name$=attribute-value ] selector is used to select elements whose attribute value ends with a specified value.
In other words, it selects any HTML elements targeted by the selector which contains an attribute name and whose attribute value ends with a value specified in attribute expression.
Example 1: 
      a[href$="tutorials"]{
        border: 2px solid red;
      }
here in this example, it selects an anchor element that has an attribute name of href and it applies a border of red color if the attribute value ends with "tutorials" Example 2: 
      div[class$="test"]{
        background-color:yellow;
      }
here in this example. it selects a div tag which has a class attribute and style in yellow color if the attribute value ends with "test".

[attribute-name*="attribute-value"]: Star sign indicates "contains"


The [attribute-name*="attribute-value"] selector is used to select elements whose attribute value contains a specified value in attribute expression.
In other words, the attribute value contains a term as "only value, a value in a list of values or as part of another value"
Example 1: 
      img[alt*="art"]{
        background-color:purple;
      }
It selects images with the alt text "abstract art" and "athlete starting a new sport" because the value "art" is in the word "starting" as well
Example 2: 
      a[href=".com"]{
        background-color:pink;
      }
It selects an anchor tag that has a hyperlink href and it contains ".com" will be styled in pink color.


[attribute-name~="attribute-value"]: tilde indicates "value in a space separated list"

The [attribute-name~="attribute-value"] selector is used to select elements with an attribute value containing a specified word.
In other words, Unlike the "contains" selector, this selector will not look for the value as a word fragment, as it will use the complete word as a "value is in a space-separated list"
Example 1 
      img[alt~="art"]{
        border= 2px solid red;
      }
It will style with a red border to an image with the alt text "abstract art" or "art show" but not "athlete starting a new sport" (which the "contains" selector would select).

Example 2 
      a[title~="my"]{
        border: 2px solid blue;
      }
It will style the element with blue border when an anchor tag will have a title containing the word "my"


[attribute-name|="attribute-value"]: pipe character indicates "begin with and immediately followed by (-) sign"

This selector is very similar to the "begin with" selector. Here, the selector matches the value that is either the only value or its the first in a dash-separated list of values. The [attribute-name|="attribute-value"] selector is used to select elements with the specified attribute starting with a specified value.
The value has to be a whole word, either alone, like class="top", or followed by a hyphen (-), like class="top-text", it is often used for language subcode matches.
In other words, It selects any html element targeted by selector which has an attribute name where the attribute value "begin with" a value specified in attribute expression & immediately followed by a hyphen (-) character.
Example: 
      li[data-year|="1900"]{
        border: 2px solid blue;
      }
It selects list items with an attribute name "data-year" has a value of "1900" and style it with 2px blue border.

Example: 
      a[title|="en-us"]{
        border: 2px solid green;
      }
It selects an anchor element that has a title with a value of "en-us" and applies a border of 2 px blue color.
