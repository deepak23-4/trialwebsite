Pseudo element selector




Pseudo element allows you to style a specific part of an element for example, it can be used to :

  1. Style the first letter, or line, of an element
  2. Insert content before, or after, the content of an element


Pseudo-elements are added to the selector using (::) symbol
Pseudo-elements allow us to apply styles to certain part of HTML elements and generate content dynamically at run time
Some of the widely used pseudo elements are :
  1. ::first-letter
  2. ::first-line
  3. ::selection
  4. ::before
  5. ::after

Syntax: 
      selector::pseudo-element-name{
        CSS declaration list;
      }
::first-letter:

The ::first-letter can only be applied to block-level element and is used to style the first letter of the HTML element.

The following properties apply to the ::first-letter pseudo-element:
  • font-properties
  • color-properties
  • background properties
  • margin properties
  • padding properties
  • border properties
  • text-decoration
  • text-transform
  • line-height
  • clear
  • float
  • vertical-align (only if "float" is :none)

Example: 
      p::first-letter{
        font-size:2.5em;
      }

::first-line:

The ::first-line pseudo-element can only be applied to block-level elements and is used to add a special style to the first line of a text.

The following properties apply to the ::first-line pseudo-element:
  • font properties
  • color properties
  • background properties
  • word-spacing
  • letter-spacing
  • text-decoration
  • vertical-align
  • text-transform
  • line-height
  • clear

Example: 
      .para2::first-line{
				text-decoration: underline;
				text-transform: uppercase;
				color:crimson;
				line-height: 1.5em;
				background-color: gray;
			}

::selection

The ::selection pseudo element matches the portion of an element that is selected by a user.
When we use the property ::selection it changes or highlights the color of the selected content

The following CSS properties can be applied to ::selection which are :
  • color properties
  • background properties
  • cursor
  • outline

Example: 
      .para3::selection{
				color:red;
				background-color: black;
			}

::before

The ::before pseudo-element is used to insert some content before the content on an HTML element
Syntax: 
      selector::before{
        content:"attribute value "
      }
wherein, content is a CSS property which we can use to style text or image almost any way we want.
and "attribute value" is a string, string means "sequence of characters enclosed in double quotations".
Example: 
      ul li::before{
        content:"Hello "
      }

::after

The ::after pseudo-element is used to insert content after the content on an HTML element
Syntax: 
      selector::after{
        content:" attribute value"
      }
Example: 
      ol li::after{
        content:" HTML"
      }
Example using image 

      ul li::before{
		  content:url("resize_twitter.png ");
	          margin: 10px;
