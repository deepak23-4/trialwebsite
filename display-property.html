CSS display property

The Display property decides how an HTML element behaves or it specifies the display behaviour of an element.
Display property changes the behaviour of an HTML element or we can say Display property gives the power to HTML element to behave differently from their nature.
For e.g,
it can give div a power of table so div will start to behave like a table
It gives span a power of div so now span will start to behave like a div

The number of property values which we use most of the times are :
  1. none
  2. inline
  3. block
  4. inline-block
  5. list-item

display:none;
This value hides the HTML element on the web-page, it doesn't delete the HTML content just hide it and the element below will take its place
This property comes handy when we are using javascript or while creating hover effect.

This property helps to change the behaviour of an HTML element to list item (li tag).
Since this display property is changing the behaviour of an HTML element to list items so it will behave in the exact same manner as list-item property behaves including all the properties like list-style-image, list-style-position etc.
