Transition Property




What is Transition
Transition means changing the state on an HTML element smoothly
Using CSS transition property we can move an HTML element smoothly instead of a sudden move.

There are five different transition functions we use and they are :
  1. transition-property
  2. transition-duration
  3. transition-delay
  4. transition-timing-function
  5. shorthand property

transition-property
CSS properties on which we need to apply transition properties, we can keep a comma separated css properties to transition them together
Syntax: transition-property:all/ none

transition-duration
The amount of time for which we want to apply the transition effect, it can be in seconds (s) or milli-seconds (ms), we can use more than one comma separated css properties to apply the transition effect
Syntax:transition-duration:2s

transition-delay
It causes the delay in order to show the transition effect by the number of seconds assigned to it
Syntax: transition-delay:2s

transition-timing-function
It uses certain inbuilt transition functions to make the HTML elements behave in a particular way
The inbuilt transition functions associated with transition-timing-functions are :
  1. ease : slow start then fast then end slowly
  2. linear: same speed from start to end
  3. ease-in: slow start
  4. ease-out: slow end
  5. ease-in-out: slow start then regular speed then slow end
  6. step-start: It reaches to initial step/ position as soon as it starts from where the animation should begin
  7. step-end: it reaches to final step/ position as soon as it starts to where the animation should end.
  8. steps(): we can customise the no. of steps. e.g, transition-timing-function: steps(4)
  9. cubic-bezier: we can define out own speed of transition. e.g, transition-timing-function: cubic-bezier(1,0.2,0.8,2);


shorthand property
shorthand property is used to save time and put all the other transition properties together using comma separated values
short-hand properties has to be applied in a specific format i.e, transition : property, duration, timing-function, delay
