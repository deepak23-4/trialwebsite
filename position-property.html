Position and z-index property




Purpose of Position property
CSS Position property is used to position an HTML element on a webpage/ HTML document. In order to apply CSS position property we use 5 different values they are :
  1. Static
  2. Relative
  3. Absolute
  4. Fixed
  5. Sticky

While using CSS Position property, it is compulsory to use left, right, top, bottom and z-index properties without them we cannot set position property on a webpage.
If we have not assigned any position to any element on a web document so by default its position will be static and it will not be impacted by left, right, top, bottom property

Static position
HTML elements display as they appear in the normal document flow
By default all HTML properties are set to static
As the name suggest (static) static elements cannot be moved i.e, static elements are not affected by top, bottom, left, right or z-index properties

Relative position
Relative position is used to move an element relative it its original/normal position, it doesn't matter when the HTML element is present on the document flow, the position property will work at the same place where it is applied. .

Absolute position
Absolute position is used to position an HTML element relative to the edge of its first positioned ancestor or relative to the edge of the browser's viweport
An ancestor could either be a parent element, a grant parent or a grant-grant parent
In order to position an HTML element with respect to its ancestor, we have to assign "position:relative" to the parent element or to a grant parent and "position:absolute" to the HTML element which we want to move.
If we have not assigned or positioned an ancestor than the HTML element will be positioned as per the edge of the browser's viewport.

Fixed position
Fixed position is used to pin or fix the position of an HTML element relative to the edge of the browser's viewport.
When we assign a position "fixed" to an HTML element, div gets converted from block level element to inline block,
The HTML element gets taken out of its normal position.
Since the HTML element is taken out of its normal position so some content below the nav-bar will be hidden behind the HTML element (nav-bar)
In order to get those hidden content visible again we need to use the padding-top property.


Sticky position
Sticky position is used to stick the html element to the top of the page
In case if the gap or the distance between top edge of the browser and the html element (nav-bar) becomes zero so the nav-bar should be fixed or stuck to that position
To make the position sticky along with "position:sticky" property we need to use "top:0px" to stick to the top edge of the browser.



z-index property z-index property is used to control the display of overlapping positioned HTML element.
When we are positioning an HTML element using position property they may get overlapped so in order to decide which one to display in front and which one to keep behind we use z-index property.
In order to use z-index property both the html element should be positioned using position property.
An HTML element with lower z-index value is displayed behind the HTML element with higher z-index value value
