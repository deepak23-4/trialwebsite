Animation property

The animation property helps us to animate any HTML element

In order to animate an HTML element we need to use certain CSS properties which are :
  1. animation-name
  2. animation duration
  3. animation-timing-function
  4. animation-delay
  5. animation-iteration-count
  6. animation-direction
  7. animation-fill-mode
  8. animation-play-state
  9. animation

animation-name
This CSS property is created to define or assign a name to the animation property
It should not have any space while assigning any animation name rather we can use underscore, hyphen or Camel casing convention
Syntax: animation-name:any-name

animation duration
This property is created to assign a particular time duration for an HTML element while getting animated or how long do we need to run the animation
Syntax: animation-duration:2s

Animation-timing-function
It helps us to set the smoothness to our HTML element which we are animating, it has certain properties and values which we use to make it work and they are:

  1. ease : slow start then fast then end slowly
    transition-timing-function:ease;
  2. linear : same speed from start to end
    transition-timing-function: linear;
  3. ease-in : slow start
    transition-timing-function: ease-in;
  4. ease-out : slow end
    transition-timing-function: ease-out;
  5. ease-in-out : slow start then regular speed then slow end
    transition-timing-function: ease-in-out;
  6. step-start : It reaches to initial step/ position as soon as it starts from where the animation should begin
    transition-timing-function: step-start ;
  7. step-end : it reaches to final step/ position as soon as it starts to where the animation should end.
    transition-timing-function: step-end;
  8. steps( ) : we can customise the no. of steps
    transition-timing-function: steps(4)
  9. cubic-bezier : we can define our own speed of transition
    transition-timing-function: cubic-bezier(1,0.2,0.8,2);


animation-delay
It helps us to define by how much time an animation should delay
Syntax: animation-delay:5s;

animation-iteration-count
It helps us define how many times we need to run the animation e.g, 3,4 or infinite
Syntax: animation-iteration-count:3

animation-direction
It defines how do we need to run the animation and it has differrent properties which are :
  1. normal : The animation works normally basis the animation properties used on it
  2. reverse : It starts from the end point where the animation ends & it comes back to the origin point from where the animation actually started. It works in anti-clockwise direction
  3. alternate : It starts from 0 & goes to 100 & then start from 100 & comes back to 0.
  4. alternate-reverse : It starts from reverse which means it starts from 100 and comes back to 0 then it goes from 0 to 100.
animation
This is a shorthand property for animation which helps to apply all the above 6 animation properties together.

Note: In order to use all the above mentions animation properties we use a special selector which is "@keyframes"
Syntax:
@keyframes animation-name{
from{CSS property}
to{CSS property}
}

or

@keyframes animation-name{
0%{CSS properties}
100%{CSS properties}
}
