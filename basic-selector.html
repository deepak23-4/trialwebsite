Basic selector




There are total 5 different types of basic selectors, they are:

  1. universal selector
  2. type selector
  3. class selector
  4. id selector
  5. grouping selector


universal selector
The universal selector (*) targets every single HTML elements on the page
Many developers use this trick to zero out the margin and padding.
universal selector targets every available HTML element on a web page and apply styles on them, basically it is used for normalization so that we can assign one common style to the whole webpage.
Syntax:
        *{
          background-color:blue;
        }


type selector
If you wish to target all elements on a page, according to their type rather than an id or a class, type selector also referred to as a tag selector or element selector because we use the element's HTML tag as a selector.
Example: span, em, strong, p, span, div etc
Syntax: 
        a{
          color:red;
          font-size:20px;
        }


class selector
The class selector starts with a (.) character followed by the name of the class. the class selector selects elements with a specific class attribute , class attribute can be used to identify more than one element
We can also use classes when you want your styling to apply to a group of elements. 
          .wrapper{
            background-color:blue;
            color:red;
          }

          .wrapper.notebook{
            border-color:purple;
            font-wieght:bold;
          }


id selector
id selector starts with a (#) character followed by the name of the id, the #id selector styles the element with the specified id.
id is used to identify only one element, we cannot use multiple id attribute value on one HTML element since its unique and should only be used once on element 

        .jam{
          color:red;
          background-color:blue;
        }


group selector
CSS group selector allows applying same styles to multiple HTML element, it allows us to target more than one element at a time to allow for a specific CSS properties or rules,
we can target multiple elements/ tags by assigning comma separated list of attributes
Syntax example: 
        h1, p, .header{
          background-color:blue;
          color:#fff
        }
