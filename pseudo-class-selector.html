CSS Pseudo class selector




Pseudo class is used to define a special state of an element, it can be used to :
  1. Style an element when a user put the mouse cursor over it
  2. Style visited and unvisited links differently
  3. Style an element when it gets focus


Pseudo class elements are added to the selectors using (:) symbol
Pseudo class selectors allow us to apply styles to HTML elements based on their current status or their special properties
When Pseudo class selectors are applied to an HTML element they will have different status during run time for e.g,
anchor element will have visited state, link state, hover state
input element will have enabled state, disabled state, required state, checked state etc

HTML elements will also have different properties in the DOM tree for e.g,
:first-child or last-child, first-of-type, last-of-type etc

Pseudo class selectors are divided into multiple groups depending upon their usage and they are :
  1. Pseudo class navigation
  2. Pseudo class user interface (UI) "form controls"
  3. Pseudo class structure (DOM)
  4. nth pseudo class


pseudo class navigation
pseudo class navigation is used to manipulate the behaviour of the hyperlinks on a webpage, they are link related pseudo-class selectors.
To style links appropriately, put the :visited rule after the :link rule but before the :hover and :active rule as defined by the (LVHA order) i.e, :link - :visited - :hover -:active (To remember the format easily we can call it "Love - hate rule" (LVHA))

pseudo class navigation has different property functions which come handy in different situation, they are:
  1. :link
  2. :visited
  3. :hover
  4. :active
  5. target

:link
We use :link pseudo class to target all anchor tags which have yet to be clicked on, in other words it selects all unvisited links on a webpage.
Syntax: 
         a:link {
           color:red;
         }


:visited
We use :visited pseudo class to apply specific styling to only the anchor tags on a webpage which have been clicked on or visited already
Syntax: 
        a:visited{
          color:green;
        }


:hover
It is normally triggered when a user hover over an element with a cursor (mouse pointer)
Syntax: 
        a:hover{
          border-bottom: 1px solid black;
        }


:active
An :active pseudo class applies while an element is being activated by a user i.e, when a user presses the mouse button and releases it.
In other words it selects a link while it is activated i.e, being clicked or the state of the button is pressed
Syntax: 
        a:active {
          background-color:yellow;
        }


:target
The :target pseudo class represents a unique element (the target element) with an id matching the URL's fragment (the URL's fragment is denoted by # followed by the targeted element)
Syntax: 
        a:target{
          background-color:red;
        }


Pseudo class User Interface (UI) (form controls)
It manipulates the behaviour of the form control in order to assign different styling properties, input (form) related pseudo class selectors

The CSS properties associated with form controls are :
  1. :focus
  2. :enabled
  3. :disabled
  4. :checked
  5. :required
  6. :optional
  7. :invalid
  8. :valid


:focus
As we know that "autofocus" attribute is used to highlight a particular form field or to attract user's attention by blinking cursor in a particular form field section, apart from that it is also helpful for people who use keyboard navigation to get to a particular form section so it highlights and user will know which section of the form he is on.
syntax example: 
        input:focus{
          color:blue;
        }


:enabled
It selects inputs that are in the default state of enabled and ready to be used.
By default all the form fiends are enabled unless they are disabled by using disabled attribute, it targets all the (enabled) form fields, to make it more specific for a particular email field we can use an attribute name and value eg.,

        input[type="email"]:enabled{
         background-color:blue;
        }
        or
        input:enabled{
          color:green;
        }


:disabled
As we know that in order to make a form field disabled which means the form field will be grayed out and a user will not be able to use it to fill any detail, so using disabled property we can style the disabled form field.

Syntax and example 
        input:disabled{
          color:gray;
        }
:checked
It selects checkboxes what are selected or checked
:checked property is used to style the radio buttons and the checkboxes where we have multiple options available and it works on the principle of adjacent sibling selector (+).
Syntax and example 
        input:checked+label{
          background-color:red;
        }
:required
It selects inputs with the required attribute.
required property is used in order to make sure that a user while filling up the form details also fill up the particular form field which we have marked as required and the form cannot be submitted without filling up the required form section and in order to style the required section in the form field we use :required property.
Syntax or Example: 
        input:required{
          background-color:purple;
        }
:optional
It selects inputs that do not have the required attribute.
optional attribute is opposite to required attribute, which means that if we have not assigned any required attribute for a form field so for all the form field section it will be considered as optional and the style will be applied accordingly.
Syntax and Example: 
        input:optional{
          background-color:red;
        }
:invalid
The :invalid CSS pseudo-class represents any <input> or other <form> elements whose content fails to validate.
As we know that recently in HTML5 there have been centain attributes added which helps to validate the values entered in the form and in order to highlight the invalid entries made in the form we can use the :invalid attribute
Syntax and Example: 
        input:invalid{
          background-color:pink;
        }
:valid
The :valid CSS pseudo-class represents any <input> and <form> elements whose content validates successfully. It is usefull for highlighting correct fields for the user, unless if we enter an invalid entry in the form field it will keep all the form field sections in the valid state.
Syntax and example: 
        input:valid{
          background-color:blue;
        }

Pseudo class structure (DOM) They are the position based pseudo class selector with in the DOM tree.Its also called as structured pseudo class selectors.
Structured pseudo class selectors meaning it is used to style content based on its relationship with parent and sibling selector
In total there are 8 properties associated with Pseudo class structure and they are:
  1. :root
  2. :empty
  3. :first-child
  4. :last-child
  5. :only-child
  6. first-of-type
  7. last-of-type
  8. only-of-type

:root
The :root selector matches the document's root element, In HTML, the root element is always the html element.
It targets the root element which is always the opening and closing of <html> and </html> tag which is by default the root selector for all HTML elements and it is used to apply styles collectively. 
        :root{
          background-color:pink;
          color:white;
        }
:empty
The :empty selector matches every element that has no children (including text contents),
It selects every HTML elements targeted by the selector if the elements has no content it will style that element.
Syntax and example; 
        p:empty{
          border: 2px solid red;
        }
:first-child
:first-child selector is used to select the first child of its parent element. if the condition meets say for example we have a parent element <div> tag and with in <div> tag we have h1 then h2 and then p tag so if we want to target any p tag if its the first child of the parent element (<div>) so the style should be applied on it but as we know the first child is h1 and p is the third child so the condition is not going to fulfil and style will not be applied
Syntax and Example 
        p:first-child{
          background-color:pink
        }
:last-child
:last-child selector is used to select the last child of its parent element. It the condition meets for the last child it applied the style and if the condition doesn't meet then the style will not be applied.
Syntax and example: 
        h1:last-child{
          border: 2px solid green;
        }
:only-child:
only-child selector is used to select the only child of its parent element, say for example we have two <p> tags with in a div1 element and within div2 element there is only one <p> tags so the condition will be met with div2 element as it has only 1 child element which is <p> tag.
Syntax and example: 
        p:only-child{
          border: 2px solid red;
        }
:only-of-type:
The differemce in between :only-child selector and only-of-type selector is that in only-child selector if in parent element say we have <div1> tag we have 3 <p> tags element so it doesn't fulfil the condition there has to be only one <p> tag as a child element similarly for only-of-type selector if there is one <div2> tag and it has 3 <p> tags and one <h1> tags so it will fulful the condition since apart from 3 <p> tags there was one <h1> tag is there and that's the :only-of-type selector which will be styled
syntax and example: 
        h1:only-of-type {
          border: 3px dotted green;
        }
:first-of-type:
:first-of-type selector will target the first of its kind regardless of if the are more then one of the same elements or if there is a different element above the targeted element as long as its the first of its type it will be styled
The difference in between first-child selector and the first-of-type selector selector is that in first-child selector if we have 1 <h1> tag above 3 <p> and we want to target first-child selector for <p> tag so it will not fulfil the condition since we have <h1> above <p> tag however in case of first-of-type selector it doesn't matter it will still style the <p> tag the one which is just below the <h1> because that's the first <p> tag of its parent element. Syntax and example: 
        p:first-of-type{
          background-color:blue;
        }
:last-of-type:
:last-of-type selector works just as first-of-type selector but the only difference is as the name suggest last-of-type selector so it selects any HTML element targeted by the selector if it is the last-child of its type in its parent HTML element
Example: 
        p:last-of-type{
          background-color:green;
        }

nth pseudo class
Its a number based pseudo class selectors
There are 6 pseudo class selectors that we use in nth pseudo class selectors and they are :
  1. :nth-child()
  2. :nth-last-child()
  3. :nth-of-type()
  4. :nth-last-of-type()
  5. :not()
  6. :lang()

Syntax: 
        :nth-child(argument){
          CSS property value;
        }


:nth-child(n)
It functions just like the first-child, As we know that if we have to target the first child element of its parent element we can use :first-child selector similarly if we want to target last child of its parent element so we can use :last-child selector but what if we want to target the second child or the third child or the fourth child so in that case we use :nth-child(n) selector.

The :nth-child(n) selector matches elements based on their position in a group of siblings,
Here, "n" could be a number, keyword or a formula wherein,
number could be anywhere starting from 1
keyword could be either even or odd
formula would be a mathematical formula that is (an+b)

Example: 
        p:nth-child(3){
          border: 2px solid green;
        }
If we have multiple <div> tag eg, div1, div2 and div3 and we only want to target <p> tags from div1 and not div2 and div3 so in that case we can write it like this, : 
        div1 p:nth-child(3){
          border: 2px solid green
        }
nth-child(even/odd)
It is entirely different from the :first-child or the first-of-type as It allows us to target multiple elements and apply style on them according to their position as child element with in the parent element, using keywords, it allows us to target all children element which are placed at the even position with in the parent element, similarly, it allows us to target all children element which are placed at the odd position with in the parent element.
Example for even: 
        p:nth-child(even){
          border: 2px solid blue;
        }
Example for odd: 
        p:nth-child(odd){
          border: 2px solid green;
        }
nth-child(an+b)
It selects any HTML element targeted by the selector if its the (an+b)th child element in its parent html element.
where, the mathamatical formula (an+b):
a represents a cycle size, which is an integer (default is 1)
n represents counter or incremental value (starts at 0),
b represents offset value, which is an integer (default is 0)
So basis the default value the formula represents: 
        :nth-child(1*n+0)
         which is equivalent to
        :nth-child(n)
In order to calculate the values for the formula (an+b)

:nth-child(1n+0)
Let's take the value of a = 1, n = 0 and b = 0
(1n+0) = 1 * 0 + 0 = 0
(1n+0) = 1 * 1 + 0 = 1
(1n+0) = 1 * 2 + 0 = 2
(1n+0) = 1 * 3 + 0 = 3

So it will target all the child element since it is incremented by 1

Similarly if we are to calculate (2n+0) so as per the formula

:nth-child(2n+0)
The value will become a = 2, n = 0 (incremented) and b = 0
(2n+0) = 2 * 0 + 0 = 0
(2n+0) = 2 * 1 + 0 = 2
(2n+0) = 2 * 2 + 0 = 4
(2n+0) = 2 * 3 + 0 = 6

So it will target all the even positioned children

Syntax: 
        p:nth-child(2n+1){
          border: 2x solid purple;
        }
It will target all children element if they are at (2n+1)th positioned in the parent element which is basically the odd positions with in the parent element

:nth-last-child()
It helps us target an HTML element which we can target from the bottom or the last, the :nth-last-child(n) selector matches every element that is the nth child, regardless of type, of its parent, counting from the last child
In other words the :nth-last-child selector matches elements based on their position among a group of siblings, counting from the end.
Example: 
        p:nth-last-child(3){
            border: 2px solid green;
        }
        It targets any HTML element if its the 3rd last child of its parent element.
:nth-of-type()
The :nth-of-type selector is similar to :first-of-type selector since it works in the same manner for example if we have 2 <div> tags which are <div1> tag and <div2> tag and in <div1> tag we have <h2> tag followed by 3 <p> tags now if we want to style 2nd <p> tags so we can use :nth-of-type selector to style the particular element.
Syntax: 
        :nth-of-type(){
          CSS declaration
        }
When we pass a number to (n) pseudo class, a pseudo class selector will target a single element from its parent HTML element.
Example: p:nth-of-child(3){ border: 2px solid blue; } if we want to target more than one element or multiple elements so in that case we can use a keyword (even/odd) .
Example: p:nth-of-type(even){ border: 2px solid green; } 
        p:nth-of-type{
          border: 2px solid blue;
        }
nth-last-of-type() The :nth-last-of-type selector works just the way as :nth-of-type selector works but the only difference is :nth-last-of-type selector start counting from the bottom.
Example: 
        p:nth-last-of-type{
          border: 2px dotted purple;
        }

:not()
The :not() selector is also known as negation pseudo class selector,
The negation pseudo class selector represents elements that do not match a list of selectors. Since it prevents specific items or selectors being selected, it is also known as negation pseudo class.
Example: 
      :not(p){
        background-color:blue;
      }
      It means set a background color for all the element that are not a <p> tag
:lang()
The :lang() selector matches elements based on the language they are determined to be in.
It is also known as language pseudo class selector, it helps us to select any HTML element targeted by the selector, if it has the lang attribute value set to the specified language code.
We can find the two letter language code by searching ISO language code on a web browser.
Syntax:
      selector:lang(ISO language code){
        background-color:yellow;
      }
Example: 
      p:lang(en){
        background-color:yellow
      }
