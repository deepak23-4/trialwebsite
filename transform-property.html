CSS Transform 2D and 3D property




Transform 2D property
There are 12 property values we have in Transform 2D property and the are; 
    1.  Translate (x), 2. Translate (y), 3. Translate (x,y)
    4.  Scale (x),     5. Scale (y),     6. Scale (x,y)
    7.  Skew (x),      8. Skew (y),      9. Skew (x,y)
    10. Rotate (angle) 11. matrix(n,n,n,n,n,n), 12. none

  1. Translate : changing position of an html element in clockwise/ anti-clock-wise direction and to give an HTML element a depth
  2. Scale : changing size of an html element
  3. Skew : changing slant of an html element
  4. Rotate : changing orientation of an html element


SYNTAX:
transform:one or more transform functions + perspective functions;
Here by one or more transform functions means, we have 1D, 2D, 3D transform functions
By perspective functions we can say "how far a user is from the computer's screen basis that we can change values to make an impactful 3D html element"


For 1 axis (like x or y axis) we use 1D function
For 2 axis (like x and y axis) we can 2D functions
For 3 axis (like x,y,z axis) we have has 3D functions
To move an HTML element horizontally, we use x axis
To move an HTML element vertically, we use y axis
To move an HTML element front or back (3D), we use z axis
We can use the values in (+) positive as well as (-) negative
To move an HTML element left of the original spot in x axis we use negative
TO move an HTML element up of the original spot in y axis we use negative


Translate value
This property is used to change the position of an HTML element in 1D, 2D or 3D function,
Syntax: transform:translate(x), transform:translate(x,y), translate3D(x,y,z);

Rotate value
The property value "rotate" is used to rotate an HTML element in order to transform it or change it to a different angle.
The degree specified while changing the angle for an HTML element could either be in positive (+) to rotate in clockwise direction or it could be in negative (-) to rotate in anti-clockwise direction.
Syntax : transform:rotate("n"deg);
Syntax example : transform:rotate(45deg);

Scale property The property scale is used to change the size of an HTML elements,
We can change the size of an HTML element in x-axis as well as y-axis,
Syntax: transform:scale(x,y);
Syntax: transform:scale(2,1);




skew property Skew property is used to change the slant of an HTML element
Syntax: transform:SkewX(x), Transform:skewY(y);
Syntax: transform:skewX(45deg);



Multiple transform functions We can use multiple transform functions on one HTML elements using a space separated list of transform functions
Syntax: transform:translateX(x) Rotate(deg) scaleY();
Example: transform:translateX(25%) Rotate(45deg) scaleY(2);

Transform-origin property The transform-origin property is used to change the position from the pivot point of an HTML element
This property helps us to change or transform the position of pivot point to translate, rotate and scale an HTML element.
Pivot point is an imaginary point, from where we use to transform an HTML element in different directions

Syntax: transform-origin:x-offset y-offset z-offset;
where,
x-offset could use px / %, it works horizontally, it uses left, center, right
y-offset could use px / %, it works vertically, it uses top, center
z-offset must use px only 


        left = 0%  center = 50%  right = 100%
        top = 0%  center = 50%   bottom = 100%
By default Pivot point is placed at the center of an HTML element
transform-origin:center center 0px;
or
transform-origin:50% 50% 0px;
Always an HTML is translated, rotated or scaled with respect to its pivot point
Using transform-origin property we can define the pivot point where ever we want
